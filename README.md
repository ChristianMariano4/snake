# Snake Project 🎮
Welcome to the Snake Project! 
\
This project is focused on learning and exploring the SDL (Simple DirectMedia Layer) library for game development and multimedia applications.

## Features ✨
- SDL Window & Rendering 🎨
- Event Handling 🎮
- Basic Animations 🔄

## Requirements 🛠️
Make sure you have the following installed:
- SDL2
``` bash
sudo pacman -S sdl2 (archlinux)
sudo apt-get install libsdl2-dev (ubuntu)
```
- SDL2_font
``` bash
sudo pacman -S sdl2_ttf (archlinux)
sudo apt-get install libsdl2-ttf-dev (ubuntu)
```
- A C/C++ compiler
